# Dank App Launcher

A premium, glassmorphic desktop launcher widget for DankMaterialShell (DMS).

## Features
- **Automatic App Discovery:** Automatically scans system applications from standard `.desktop` directories (`/usr/share/applications/` and `~/.local/share/applications/`).
- **Live Fuzzy Search:** Instant fuzzy search matches both app names and executables.
- **Normalized Category Filtering:** Filters applications with categorized chips (Internet, Development, Game, Graphics, Office, Multimedia, System, Utilities).
- **Responsive Layout:** Automatically adjusts columns grid layout based on your widget size.
- **Glassmorphic Theme Integration:** Adapts dynamically to system colors, margins, outline opacity, and animations.
